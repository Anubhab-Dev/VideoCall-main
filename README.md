# 🎥 Video Call Web Application

A real-time video calling web application that enables seamless communication between users using modern web technologies. This project demonstrates peer-to-peer video streaming, responsive UI design, and efficient real-time data handling.

---

## 🚀 Features

* 📹 Real-time video calling
* 🎙️ Audio communication support
* 🔗 Peer-to-peer connection (WebRTC)
* ⚡ Fast and responsive UI
* 🌐 Cross-browser compatibility
* 🔒 Secure communication

---

## 🛠️ Tech Stack

**Frontend:**

* React.js
* HTML5, CSS3, JavaScript

**Backend:**

* Node.js
* Express.js

**Real-time Communication:**

* WebRTC
* Socket.io

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/Anubhab-Dev/VideoCall-main.git
cd VideoCall-main
```

### 2️⃣ Install dependencies

**For client:**

```bash
cd client
npm install
```

**For server:**

```bash
cd ../server
npm install
```

### 3️⃣ Run the application

**Start backend:**

```bash
npm start
```

**Start frontend:**

```bash
cd client
npm start
```

---

## 🌍 How It Works

* Users join a room
* WebRTC establishes peer-to-peer connection
* Socket.io handles signaling between users
* Video/audio streams are shared in real-time
